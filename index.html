Projects

Read&See

This is a project where we created a webapplication using JavaScript, HTML and CSS together with React.js. We used two different API's to retrieve data about books and movies and we used Firebase to store user data as well as data about adaptations. It was built as a part of the course DH2642 Interaction Programming and the Dynamic Web at KTH.

Read&See is a place where you can find out if your favorite books have been adapted to film! You can create an account, and then you will be able to add movies to your watchlist!
You can also add connections between books and movies that does not appear to be adaptions on the site but actually are.





inWinetation

In this project we used Firebase Google Authentication, Cloud Firestore, Onsen UI, React-Google-Maps/API and CSS. The project was created as a part of the course DM2518 Mobile Development with Web Technologies at KTH.

inWinetation is a mobile application used to find your friends on a map and perhaps join them for a beverage. You can add messages for your own position to let your friends know what you are up to.





B3

DM2601 Media Technology and Interaction Design Through a design process we used several different methods in order to design a new functionality for an application like Spotify. The purpose we had in mind was how listening to music could be a group activity during a pandemic with restrictions. Through the phases of Discover, Define, Develop, and Deliver we used the methods body- and brainstorming, interviews, literature research, creating personas, moodboard, lo-fi and hi-fi sketches, and an interactive prototype made in figma. A workbook of our process was also made in figma.
